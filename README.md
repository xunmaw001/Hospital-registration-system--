# Hospital-registration-system--
这款基于微信小程序的医院挂号系统，是便捷就医的好帮手。无需下载，打开微信就能使用。系统整合多家医院信息，科室分类清晰，医生简介详尽，出诊时间、号源情况一目了然。患者可依自身需求，快速筛选科室与医生，在线完成挂号操作，支持微信支付，安全便捷。还能设置就诊提醒，避免错过看病时间。就医后可对就诊体验评价反馈。简化挂号流程，让患者轻松就医 。
